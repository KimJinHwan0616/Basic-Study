># VCS *(Version Control System)*
>프로젝트 `버전`을 관리하는 `소프트웨어`
>### 깃, 깃 호스팅 서비스
---

## 깃 *(git)*

+ ### 시간
  
  >커밋 *(commit)*
  >```
  >`프로젝트`를 `버전`으로 `저장`하는 것
  >```
  >
  >리셋 *(reset)*
  >```
  >`프로젝트`를 `이전 버전`으로 되돌리는 것
  >```
  >
  >리버트 *(revert)*
  >```
  >`프로젝트`를 `특정 버전`으로 되돌리는 것
  >```

+ ### 공간
  
  >브랜치 *(branch)*
  >```
  >`특정 버전`을 `복사`하는 것(≒사본)
  >```
  >
  >머지 *(merge)*
  >```
  >`브랜치 버전`을 `원본 버전`으로 가져오는 것
  >```
