## 서버 *(server)*
`데이터`를 전달 `하는` 컴퓨터

## 클라이언트 _(client)_
`데이터`를 전달 `받는` 컴퓨터

---

## 데이터 센터 *(IDC, Internet Data Center)*
여러 개의 `서버`를 `한 곳`에서 관리하는 시설

+ ### 온프레미스 *(On-Premise)*
  `내부`에서 `서버`를 `구축`하는 것

+ ### 서버 호스팅 *(Server Hosting)*
  `외부(데이터 센터)`에서 `서버`를 `대여`해주는 것

+ ### 클라우드 컴퓨팅 서비스 _(Cloud Computing Service)_ ★
  `인프라`, `플랫폼`, `소프트웨어`를 `클라우드(인터넷)`로 `대여`해주는 서비스
  ```
  예) AWS(아마존), GCP(구글), NCP(네이버)
  ```
  
  >IaaS _(Infrastructure As A Service)_: `인프라`
  >```
  >인프라: 가상화, 서버, 저장소, 네트워크
  >
  >예) AWS - EC2
  >```
  >
  >PasS _(Platform As A Service)_: `플랫폼`
  >```
  >플랫폼: '인프라' + 런타임(Runtime), 미들웨어(Middleware), 운영체제(O/S)
  >
  >예) 헤로쿠, AWS - 일래스틱 빈스토크
  >```
  >
  >SaaS _(Software As A Service)_: `소프트웨어`
  >```
  >소프트웨어: '플랫폼' + 에플리케이션, 데이터베이스
  >
  >예) 구글 드라이브, 네이버 N클라우드, 드롭박스, MS 오피스 365
  >```
