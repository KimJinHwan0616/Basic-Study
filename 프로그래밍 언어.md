## 고수준 언어
자연어(사람의 언어)에 가까운 언어

+ ### 자연어 *(한글, 영어)*

+ ### 고수준 프로그래밍 언어 ★
  >컴파일 언어
  >```
  >코드 전체를 기계어로 바꿔서 실행 파일을 만들어서 실행하는 언어
  >
  >예) C, C++, C#, 자바, 타임스크립트, 코틀린, 스위프트, 스칼라, Go, 러스트
  >```
  >
  >인터프리터 언어
  >```
  >코드를 읽으면서 바로 해석해서 실행하는 언어
  >
  >예) 자바스크립트, 파이썬, 루비, PHP, 펄, R
  >```

## 저수준 언어
기계어에 가까운 언어

+ ### 어셈블리어
  기계어를 사람이 이해할 수 있도록 알파벳, 숫자로 표현한 언어

+ ### 기계어
  0, 1로 이루어진 언어
