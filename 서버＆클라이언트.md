## 서버
데이터를 전달 `하는` 컴퓨터

## 클라이언트
데이터를 전달 `받는` 컴퓨터

---
## 데이터 센터(IDC)
대량의 서버들을 `한 곳`에서 안정적으로 관리하는 시설(internet data center)

+ ### 온프레이스
  `내부`에서 서버를 `구축`하는 것

+ ### 서버 호스팅
  `외부(데이터 센터)`에서 서버를 `대여`해주는 것

+ ### 클라우드 컴퓨팅 서비스 ★
  인프라, 플랫폼, 소프트웨어를 `클라우드(인터넷)`로 제공하는 서비스
    ```
    - 인프라: 가상화, 서버, 저장소, 네트워크
    - 플랫폼: '인프라' + 런타임(Runtime), 미들웨어(MIddleware), 운영체제(O/S)
    - 소프트웨어: '플랫폼' + 에플리케이션, 데이터베이스 

    예) AWS(아마존), GCP(구글), NCP(네이버)
    ```
  >IaaS: `인프라` 제공
  >```
  >예) AWS - EC2
  >```
  >
  >PasS: `인프라` + `플랫폼` 제공
  >```
  >예) 헤로쿠, AWS - 일래스틱 빈스토크
  >```
  >
  >SaaS: `인프라` + `플랫폼` + `소프트웨어` 제공
  >```
  >예) 구글 드라이브, 네이버 N클라우드, 드롭박스, MS 오피스 365
  >```
