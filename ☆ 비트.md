># 비트 *(bit, Binary Digit)*
>`데이터` 최소 단위 *(0 또는 1)*
>
>### 정수, 실수, 불, 유니코드 문자
바이트 *(byte)*: 8비트
```angular2html
예) 
n bit = 2ⁿ개
1 byte = 8 bit = 256개
```
---

## 정수
```
byte(8 bit): -2^7 ~ 2^7-1
short(16 bit): -2^15 ~ 2^15-1
int(32 bit): -2^31 ~ 2^31-1
long(64 bit): -2^63 ~ 2^63-1
```

## 실수
```
float(32 bit): 소수점 6 ~ 7자리까지
double(64 bit): 소수점 15자리까지
```

## 불
```
boolean(1 bit)
```

## 유니코드 문자 *(Unicode Character)*
전 세계 `모든 문자`
```
char(16 bit)
```

+ ### 문자 인코딩 *(Character Encoding)*
  `유니코드 문자` → `어셈블리어`
  ```
  예) UTF-8
  ```
